PROJECT TITLE: 3D Asteroids?

HOW TO START THIS PROJECT: Launch the asteroides main class. Also try out the other main classes too.
AUTHORS: David Neufeld
USER INSTRUCTIONS: In the asteroids game:
WSAD to move
Space+LShift to move up and down
Up Arrow to shoot
Q + E to look left and right
B to look behind you

In flight class use arrow keys rather than wsad
